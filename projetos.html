locator-pet

Desenvolvimento de um website com Django (Python) com o intuito de localizar animais perdidos. Através do sistema é possível criar posts que ficarão disponíveis para outros usuários visualizarem e entrarem em contato caso tenham alguma informação relevante. O sistema possui autenticação e utiliza a maioria dos recursos disponíveis do Django, inclusive a comunicação através do Django Rest Framework, disponibilizando uma API para uso e consumo.
